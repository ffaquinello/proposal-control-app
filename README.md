# ProposalControlApp

Controle de Propostas Comerciais.

Funcionalidades Principais: Na empresa onde trabalho existe a necessidade de gerar propostas comerciais para cada requisito Solicitado por diversas áreas. 
A ideia é ter o controle do status da propostas (elaboração, enviada ao cliente, aprovada, rejeitada, em execução, concluída) além disso o formulário
de entrada deve permitir informa a empresa para qual esta sendo gerada a proposta, o nome do responsável pela solicitação, o Nome utilizado pela proposta e
uma breve descrição do que sera realizado.
